# Screenshots

Add app screenshots here for the portfolio README and demo, e.g.:

- `dashboard.png` - live telemetry dashboard
- `pump-control.png` - manual pump control + emergency stop
- `automation.png` - thresholds + automation decision
- `alerts.png` - alerts list
- `history.png` - historical charts
- `diagnostics.png` - Java native module network diagnostics

To capture from an Android emulator:

```bash
adb exec-out screencap -p > docs/screenshots/dashboard.png
```

Then reference them from the root `README.md`.
